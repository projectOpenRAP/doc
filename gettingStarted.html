Getting Started

Last updated: February 16, 2018

Get a Device

OpenRAP has been tested on these devices:

  • Raspberry Pi 3 Model B
Newer devices will be added as and when adequate testing confirms their compatibility with our product.

Install Base Image

Steps to follow:

  1. Download the zip file from here.
  2. Extract the image from the zip file.
  3. Burn the extracted base image into a microSD Card with a minimum capacity of 4GB.
  4. Insert the SD Card into the device and boot it.
  5. You should see a Hotspot published in 10 seconds with the name "OpenRAP"
  6. Build the latest version of OpenRAP and upgrade your device by following the steps mentioned below.

Build the Latest Release

The following steps let you build the latest version of OpenRAP to use in your box.

Clone our Repository

Clone the code from our repsoitory into your local machine.

Using SSH:

git clone git@github.com:projectOpenRAP/OpenRAP.git

Using HTTPS:

git clone https://github.com/projectOpenRAP/OpenRAP.git

Ensure you have the prerequisites

Run the following commands:

node -v
npm -v
go version

The output should be the corresponding version numbers of NodeJS, NPM and GoLang installed. If you need to install them, please check out the NodeJS, the NPM and GoLang download and documentation pages.



Warning!

This project uses NodeJS v.8.9.4 and NPM 5.6.0. Ensure that this is the minimum version in your machine as well. Using versions earlier than this can lead to unexpected behavior or even failure of execution.

Install Packages

Install the prerequisite packages by:

cd OpenRAP/devmgmtV2
npm install
cd ../devmgmtui
npm install

Building the Image

Run the script that builds the .tgz file by:

cd .. && python buildimage.py

The python script also takes positional arguments that allow you to modify the output based on the board and platform you wish to use. These positional arguments are:

  • --board Defines the board to be used
  • --platform Defines the platform to be used
  • --profile Defines a custom profile
  • --clean Specifies the type of image to be cleaned

The positional arguments can be accessed in the terminal with the -h argument.

Verifying Image Creation

The tgz file will be named as openrap-<version-no>.tgz and stored in the build/ directory.

Upgrade Base Image to Latest Version

The following steps will allow you to upgrade your device to the latest version.

  1. Connect to the hotspot and then to the device at admin.openrap.com:8008
  2. Use root for the username and password.

    Alert!

    Make sure you modify your root password and [preferably] create another account with limited priveleges for regular use after setup.

  3. Click on the upgrade button and select "local files"
  4. Navigate to the built image and upload it
  5. The upgrade will begin once the image is uploaded.
  6. Once you get the notification of a successful upgrade, reboot the device.
  7. The device is now in the latest version.
